# arduino-environmental-logger

This is some test code to make a low-power Arduino-based environmental data logger.

## Requirements

### Software

1. PlatformIO

### Hardware

1. Arduino Uno
2. Robodyn Data logger shield (or similar)
3. DHT11 Sensor (1-wire connected to Arduino pin 4)
